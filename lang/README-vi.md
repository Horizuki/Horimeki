# Horimeki (ホリメキ)
Horimeki là selfbot giúp bạn có thể treo/giữ voice 24/7.\

## Yêu cầu
> **Cần Node.js 20.18.0 hoặc phiên bản mới hơn để sử dụng.**

## Cài đặt

```bash
git clone https://github.com/Horizuki/Horimeki
cd Horimeki
npm install
```

## Chạy

```bash
node index.js
```
